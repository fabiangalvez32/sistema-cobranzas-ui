# 🚀 CobranzasPro UI

Un prototipo avanzado de Gestión de Cobranzas diseñado con un enfoque radical en la **Experiencia de Usuario (UX)**, la **Tecnología Persuasiva** y la **Psicología Computacional**. 

Este proyecto demuestra cómo el diseño moderno puede transformar una herramienta interna tradicional (pesada, lenta y con alta carga visual) en un sistema rápido, intuitivo y agradable de usar.

---

## 🧠 Principios UX Aplicados

Este proyecto no es solo una "cara bonita". Cada botón, espacio y flujo de trabajo está respaldado por los siguientes pilares y heurísticas de UX:

### 1. Útil (Useful): Reducción de Fricción Operativa
- **Resultados Rápidos (1-Clic):** En lugar de menús desplegables interminables, las acciones más comunes (Promesa de pago, Buzón, etc.) son botones grandes de un solo clic. Esto ahorra cientos de clics por usuario al día.
- **Atajos de Teclado:** Presionando `Ctrl + Enter` se guarda la gestión automáticamente, permitiendo a los *Power Users* trabajar sin tocar el mouse.

### 2. Usable (Usable): Reducción de Carga Cognitiva
- **Divulgación Progresiva (Progressive Disclosure):** Pasamos de una interfaz saturada a un diseño limpio de 2 columnas. Los datos secundarios del cliente (como dirección física o correos alternativos) están ocultos por defecto y solo se muestran cuando el gestor hace clic en "Ver más detalles".
- **Valores por Defecto (Smart Defaults):** El sistema pre-selecciona automáticamente el teléfono principal y asume el tipo de operación para agilizar la gestión.

### 3. Accesible (Accessible): Diseño Inclusivo
- **Modo Oscuro (Dark Mode):** Interruptor integrado para cambiar a un tema oscuro, diseñado para reducir la fatiga visual de los gestores que pasan turnos de 8 horas mirando la pantalla.
- **Codificación Doble:** Los estados no solo usan color (ej. rojo para deudas, verde para pagos), sino que se acompañan de íconos claros para garantizar la usabilidad en personas con daltonismo.

### 4. Deseable (Desirable): El "WOW Factor"
- **Sugerencias de Inteligencia Artificial:** Tarjetas predictivas que le dicen al gestor a qué hora es más probable que el cliente responda.
- **Micro-animaciones de Recompensa:** Al registrar un éxito (ej. Promesa de pago), el sistema lanza una micro-animación de celebración, liberando dopamina (gamificación) y manteniendo al empleado motivado.

## 🛠️ Tecnologías Utilizadas
- **Next.js / React** (App Router)
- **Tailwind CSS** (Para estilos y utilidades rápidas)
- Diseño Responsivo y Componentes Modulares

## 🚀 Cómo ejecutarlo localmente

1. Clona este repositorio:
```bash
git clone https://github.com/fabiangalvez32/sistema-cobranzas-ui.git
```

2. Instala las dependencias:
```bash
npm install
```

3. Inicia el servidor de desarrollo:
```bash
npm run dev
```

4. Abre tu navegador en [http://localhost:3000](http://localhost:3000). Para ver la versión final súper optimizada, dirígete a [http://localhost:3000/optimizada](http://localhost:3000/optimizada).
